log function

The log function output a message to the web console.

Example


#[wasm_bindgen(start)]
pub fn main() -> Result<(), JsValue> {
let array = js_sys::Array::new();
array.push(&"Hello Console Log".into());
web_sys::console::log(&array);
Ok(())
}

Syntax


                pub fn log(data: &Array)
